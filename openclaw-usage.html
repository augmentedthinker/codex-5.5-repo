How to use this page

  • Public-safe by default: this page shows only manually entered percentages, not tokens, credentials, account IDs, or raw provider responses.
  • Manual update flow: ask Ash to check current usage, then update these demo gauges to the real percentages and push the page.
  • Future upgrade path: if we want live telemetry later, keep the live data private/local and publish only a sanitized snapshot.
