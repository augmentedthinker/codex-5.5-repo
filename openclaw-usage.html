How to use this page

  • Source: Ash uses OpenClaw’s private provider-usage loader for openai-codex, which reads the OAuth-backed Codex usage snapshot from the ChatGPT usage backend. Raw tokens, account IDs, and provider responses are not published here.
  • Window math: the source provides used percentage, reset time, and window duration. The displayed start time is calculated as reset time − window duration; the displayed end time is the provider reset time. The secondary green ring shows elapsed time in the current window, so usage can be compared against pace.
  • Update flow: ask Ash for a “usage dashboard update.” Ash should check current usage, update the gauges, start/end labels, and last-updated pill, commit the change, push it to GitHub, and verify the live page.
